## Комментарии

Разработано веб-приложение на **Flask**, которое позволяет выполнять асинхронный поиск научных публикаций в базе данных **Crossref API** (крупнейший архив научных статей, DOI и метаданных).

Пользователь вводит имя автора или название статьи, и приложение отправляет асинхронный HTTP-запрос к API Crossref, получает результаты и отображает их в удобном карточном интерфейсе с адаптивной версткой на **Bootstrap**.

### 1. Бэкенд на Flask
- Создан маршрут `GET /` для отображения HTML-формы поиска.
- Создан маршрут `POST /search` для обработки запросов от пользователя.
- Реализована **асинхронная функция** `async_search_publications()`, которая:
  - Принимает параметры автора и названия.
  - Формирует параметры запроса к API Crossref (`query.author`, `query.title`, `rows`).
  - Использует `aiohttp` и `asyncio` для неблокирующих HTTP-запросов.
  - Обрабатывает таймауты, ошибки соединения, пустые ответы.
  - Извлекает из JSON ответа первые 10–20 публикаций.

### 2. Асинхронная обработка данных
- Функция `process_publication()` асинхронно извлекает из каждой публикации:
  - Название статьи
  - Фамилию и имя первого автора
  - Название журнала или сборника
  - Год публикации 
  - Аффилиацию (вуз/организацию) первого автора (если отсутствует — выводится «Не указано»)
- Используется `asyncio.gather()` для параллельной обработки нескольких публикаций.
